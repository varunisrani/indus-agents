# NexusAI - AI Mobile Company Website

A modern, multi-page website for NexusAI - a fictional AI mobile company specializing in artificial intelligence solutions for mobile applications.

## Project Structure

```
ai-mobile-company/
├── index.html          # Home page
├── about.html          # About Us page
├── services.html       # Services page
├── products.html       # Products page
├── contact.html        # Contact page
├── css/
│   └── style.css      # Main stylesheet
├── images/            # Image assets (empty - add your images)
└── js/                # JavaScript files (empty - pure HTML/CSS used)
```

3. **Services** (services.html)
   - AI Chatbots & Virtual Assistants
   - Vision AI & Image Recognition
   - Voice AI & Speech Processing

   - AI Security & Fraud Detection
   - Custom AI Development
   - Development process
   - Industries served

4. **Products** (products.html)
   - NexusChat SDK
   - VisionCore SDK
   - VoiceAI SDK
   - PredictML Engine
   - SecureAI Shield
   - NexusAI Core (All-in-One)
   - Pricing plans (Starter, Professional, Enterprise)

5. **Contact** (contact.html)
   - Contact form
   - Contact information
   - FAQ section (with accordion)
   - Office locations
   - Social media links

- **Animations**: Fade-in, slide-up, rotating gradient background
- **Responsive Breakpoints**:
  - Desktop: 1200px+

  - Mobile: < 768px

## How to Use

1. **Open the website**
   - Simply open `index.html` in any modern web browser
   - No build process or server required

2. **Navigate between pages**
   - Use the navigation menu in the header
   - All pages are linked together

3. **Customize**
   - Edit `css/style.css` to change colors, fonts, and styles
   - Modify HTML files to update content
   - Add your own images to the `images/` folder

4. **Deploy**
   - Upload all files to any web hosting service
   - Works with GitHub Pages, Netlify, Vercel, etc.
   - No backend required
