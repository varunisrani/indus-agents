API Reference

Complete API documentation for integrating with AI Machine

API Endpoints

  • POST /train
  • POST /predict
  • GET /models
  • GET /status
  • DELETE /model/:id

Example: Training Request

POST /api/v1/train
Content-Type: application/json

{
  "model_type": "neural_network",
  "dataset": "s3://bucket/data",
  "parameters": {
    "epochs": 100,
    "batch_size": 32,
    "learning_rate": 0.001
  }
}

Example: Prediction Request

POST /api/v1/predict
Content-Type: application/json

{
  "model_id": "model_123",
  "input": [1.0, 2.0, 3.0, 4.0]
}
