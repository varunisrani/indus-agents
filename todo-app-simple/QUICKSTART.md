## Tips & Tricks

### Speed Up Adding Todos
- Type and press Enter (no need to click "Add")
- Press `N` to focus the input from anywhere
- Leave description, category, etc. blank if not needed

### Organize with Categories
- Use different categories for different life areas
- Filter by category to focus on one area
- Color-coded badges make it easy to scan

### Use Priorities Wisely
- Set High priority for urgent tasks
- Set Low priority for "nice to have" tasks
- Sort by priority to tackle important stuff first

### Due Dates
- Set due dates for time-sensitive tasks

### Bulk Operations
- Select multiple todos with the circle icon
- Use "Complete" to mark all as done
- Use "Delete" to remove all at once
- Press "Cancel" or Escape to clear selection

## Troubleshooting

### Todos Not Saving
- Check if browser is in private/incognito mode
- Make sure you have storage space available
- Try exporting as backup immediately

### App Not Loading
- Make sure you opened `index.html`
- Try a different browser (Chrome, Firefox, Safari, Edge)
- Check JavaScript is enabled in browser settings

### Lost Data
- If you exported before, import your backup
- Check browser cache/storage settings
- Data is stored locally - clearing browser data deletes it
