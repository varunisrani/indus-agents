# AI IPO Company Website - Project Plan

## Project Overview
Build a professional multi-page website for an AI IPO (Initial Public Offering) company that helps AI startups go public.

## Tech Stack
- **HTML5** - Structure
- **CSS3** - Styling (no frameworks)
- **JavaScript (Vanilla)** - Interactivity (no React)
- **Responsive Design** - Mobile-first approach

## Website Structure

```
ai-ipo-company/
├── index.html              # Homepage
├── about.html              # About Us
├── services.html           # Services
├── ipo-pipeline.html       # IPO Pipeline
├── contact.html            # Contact
├── css/
│   └── style.css          # Main stylesheet
├── js/
│   └── main.js            # JavaScript functionality
└── assets/
    ├── images/            # Images and icons
    └── fonts/             # Custom fonts if needed
```

- Footer with navigation

### 2. About Page (about.html)

### 3. Services Page (services.html)

- Market positioning
- Investor relations
- Post-IPO support

### 4. IPO Pipeline Page (ipo-pipeline.html)
- Current pipeline companies
- Success stories
- Upcoming IPOs
- Industry sectors
- Performance metrics

### 5. Contact Page (contact.html)
- Contact form
- Office locations
- Team contacts
- Social media links
- Newsletter signup

## Design Features

### Color Scheme
- Primary: Deep Blue (#0A2463)
- Secondary: Gold/Amber (#F7931E) - symbolizing wealth/success
- Accent: Teal (#3E92CC)
- Neutral: White, Light Gray, Dark Gray

### Typography
- Headings: Modern sans-serif (Montserrat or similar)
- Body: Clean sans-serif (Open Sans or similar)
- Professional, readable, business-oriented

### UI Elements
- Smooth scroll navigation
- Hover effects on buttons and cards
- Animated statistics counters
- Responsive hamburger menu for mobile
- Sticky header
- Back-to-top button
- Form validation
- Loading animations

### Interactive Features
- Mobile navigation toggle
- Smooth scrolling between sections
- Testimonial carousel/slider
- FAQ accordion
- Contact form with validation
- Newsletter subscription
- Animated counters for statistics
- Scroll animations for elements
