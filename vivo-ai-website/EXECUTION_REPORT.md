## Customization Guide

### Change Brand Colors
Edit `css/main.css`:
```css
:root {
  --primary-color: #yourcolor;
  --secondary-color: #yourcolor;
}
```

### Change Content
Open any `.html` file and edit the text directly.

### Add New Pages
1. Copy an existing page
2. Update the content
3. Link to it from navigation
4. Add page-specific CSS if needed

### Add Images
Place images in:
- `images/icons/` - Icons and logos
- `images/products/` - Product images
- `images/team/` - Team photos
- `images/office/` - Office/culture photos
